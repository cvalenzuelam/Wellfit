# Wellfit Payments — Postman import

1. Import this folder in Postman.
2. Select env **Wellfit Payments STAGE**.

## Structure

```
Wellfit Payments
├── 1 — Core
├── 2 — Tickets
└── 3 — Regression   (CNP, TokenVault, Provisioning, Wallet, PAY-2603, Treasury)
```

Rebuild after adding sources: `python3 "Wellfit Payments/rebuild.py"`
