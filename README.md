# Wellfit QA workspace

Personal QA notes, runbooks, Postman collections, and Cursor rules for Wellfit payments work.

## Sync (Windows ↔ Mac)

```bash
git pull    # before working
git add .
git commit -m "describe changes"
git push
```

## Postman environments

Environment files (`*postman_environment*.json`) are **not** in this repo because they can hold client secrets. Copy them manually between machines or export from Postman.
