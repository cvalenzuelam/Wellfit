**AC Description:** Failed/stranded batches are not marked sent in Platform DB after SFTP transport failure.

- TC-05 failure run completed.
- TC-03 staged row markers unchanged (`PAY3821-E2E-%`).

- **All three** staged rows still have **`RequestSentTimeStamp` IS NULL**.

**Steps**

First run **SELECT** on `Payments.FundingBatches` where `BatchFileName LIKE 'PAY3821-E2E-%'`.

Then confirm **3 rows** returned.

Next verify **`RequestSentTimeStamp`** is **NULL** for every staged row.
